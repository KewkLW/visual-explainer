8 — Running Tests
How to Run

Install dependencies and run the generated Playwright tests:

cd ~/.agent/tests/kewk-fitness
npm install
npx playwright install
npx playwright test
npx playwright test --ui  # interactive mode

Visual regression baselines are generated from Stitch screen.png files, stored in tests/baselines/. Update baselines after intentional UI changes with npx playwright test --update-snapshots.

Configuration Notes

Set BASE_URL in playwright.config.ts to your dev server address. Default: http://localhost:3000. Auth tests use TEST_USER_EMAIL and TEST_USER_PASSWORD environment variables.
